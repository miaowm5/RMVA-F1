RPG::MoveRoute

移动路线的数据类。

超类

  • Object

调用者

  • RPG::EventPage

属性

repeat

是否勾选「循环执行」选项的复选框。

skippable

是否勾选「忽略障碍」选项的复选框。

wait

是否勾选「等待至移动结束」选项的复选框。

list

移动路线内容，RPG::MoveCommand 对象的数组。

定义

class RPG::MoveRoute
  def initialize
    @repeat = true
    @skippable = false
    @wait = false
    @list = [RPG::MoveCommand.new]
  end
  attr_accessor :repeat
  attr_accessor :skippable
  attr_accessor :wait
  attr_accessor :list
end
